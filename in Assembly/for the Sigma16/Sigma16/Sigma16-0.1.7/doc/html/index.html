Sigma16: a simple computer architecture assembler and emulator

  • Introduction

    • Tutorial: get started with some simple examples -- [Programming:}(file:programming.html) writing programs in assembly language
  • Reference manual
    • Architecture: explains the data representations and instruction set of the system
    • Assembly: defines the assembly language, a readable notation for writing machine language programs
    • Linking: shows how separate object modules are linked into an executable load module
    • Execution: describes how to use the emulator software to simulate the execution of a program
  • Examples
    • Add: a small program that adds two numbers
    • LoadStore: illustrates the usage of simple static variables and how to access them with the load and store instructions
    • Mult: demonstrates multiplication of integers
    • Logic: gives examples of Boolean logic operations
    • Shift: performs shifts of words
    • Write: a program that invokes the virtual operating system to perform some text output
    • Max: illustrates loops and arrays; it finds the maximum value in an array of integers
    • Factorial: a recursive factorial program that illustrates functions and the call stack
