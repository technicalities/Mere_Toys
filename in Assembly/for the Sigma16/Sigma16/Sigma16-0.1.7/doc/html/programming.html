Programming in Sigma16 assembly language

Words

Most real computers have words with several different sizes. It is common to have 8-bit bytes, 16-bit short words, 32-bit words, and 64-bit long words, but this terminology is not universal

Sigma16 has only one word size: all its words are 16 bits. They are not subdivided into smaller groupings of bits, like bytes.

Memory

The memory consists of a set of words which are accessed by address. Each word in the memory contains 16 bits. There are no bytes, long words, etc.

The architecture uses several primitive data types:

  • Binary numbers, used for addresses

  • Two's complement integers

  • Booleans, represented in a word. Zero means False, any nonzero value means True.

  • Characters, represented as 16-bit unicode.
